Trips

Trip #1

This rafting trip is a class 3 trip. It contains almost constant rapids that can be big enough to flip a raft if the raft is sideways when it hits the rapid. It is safe for people who are experienced with rafting, as long as they follow the rules.

Trip #2

This rafting trip is a class 2 trip with some occasional class 3 rapids. About two thirds of the section is rapids, but most of the rapids are safe to go over. It is safe for almost everyone to do this trip, but it is recommended that someone experienced is there to guide beginners.

Trip #3

This trip is a class 1 trip. It has almost no rapids and should be safe for anyone. It is meant for people who aren't comfortable with rafting on more dangerous rivers.
